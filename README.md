# Contact Management System – Week 3
# The Developer Arena

A Python-based Contact Management System using functions, dictionaries,
file handling, and input validation.

## Features
- Add, search, update, delete contacts
- Phone & email validation using regex
- JSON file storage
- CSV export
- Menu-driven program
- Unit testing

## Run Project
python contacts_manager.py

## Run Tests
python test_contacts.py
